# 🐿️ 한글나라 사용설명서

> **다람이랑 같이 한글을 배워요!**
> 6~7세 아동을 위한 한글 학습 웹앱 (Progressive Web App)

---

## 🚀 시작하기

### 접속 방법

**아무 인터넷 브라우저에서 아래 주소로 접속하세요:**

> **https://jaden8429.github.io/hangulnara-web/**

WiFi가 달라도 어디서나 접속 가능합니다. (집/할머니댁/유치원 등)

### 갤럭시탭/태블릿에 "앱처럼" 설치하기

1. 위 주소로 접속
2. 브라우저 메뉴 → **"홈 화면에 추가"** 선택
3. 홈 화면의 한글나라 아이콘을 누르면 전체화면으로 실행됩니다
4. **한 번 설치하면 오프라인에서도 동작합니다** (인터넷 없이 학습 가능)

## 🏠 홈 화면 한눈에 보기

```
🐿️  다람이 인사말
─────────────────────────────
🐿️ 오늘 추천: 복습부터 해볼까?  [추천 버튼]
─────────────────────────────
  📝 학습하기       🪄 글자 만들기
  🎮 듣고 맞추기    📖 문장 읽기

            👨‍👩‍👧 부모님
```

> 💡 **다람이 추천 배너**: 매번 들어올 때마다 다람이가 "오늘은 뭘 할까?"를 알려줘서 아이가 헤매지 않아요.

## 📚 학습 메뉴 6가지

### 📝 학습하기 — 기본 학습 코스

**자모부터 받침까지 한 단계씩 차근차근.**

차례대로 잠금이 풀리는 6단계 구조:

| 단계 | 내용 | 예시 |
|------|------|------|
| ✏️ 준비 | 선 긋기, 동그라미 | 가로선, 세로선, 원 |
| ㄱ 자음 | 14개 기본 자음 | ㄱ ㄴ ㄷ ㄹ ㅁ … |
| ㅏ 모음 | 10개 기본 모음 | ㅏ ㅑ ㅓ ㅕ ㅗ … |
| 가 쉬운 음절 | 받침 없는 음절 | 가 나 다 라 마 … |
| 곰 받침 글자 | ㄴ/ㅁ/ㄹ 받침 | 산 곰 달 별 … |
| ㅐ 이중모음 | 합성 모음 | 애 에 와 워 의 … |

**각 글자는 5단계 흐름으로 학습합니다:**

```
👂 LISTEN  →  👀 GUIDED  →  ✍️ TRACE  →  ✏️ FREE  →  ⭐ PRAISE
 듣기        획순보기      따라쓰기      혼자쓰기       칭찬
```

1. **👂 듣기 (LISTEN)** — 다람이가 글자 이름과 소릿값을 알려줘요
   - 자음은 "기역"(이름) + "[그]"(소릿값) 둘 다 들려줍니다
2. **👀 보기 (GUIDED)** — 획순 애니메이션으로 어떻게 쓰는지 보여줘요
3. **✍️ 따라쓰기 (TRACE)** — 흐릿한 가이드 위에 손/펜으로 따라 그려요
4. **✏️ 혼자쓰기 (FREE)** — 가이드 없이 빈 칸에 혼자 써봐요
5. **⭐ 칭찬 (PRAISE)** — 별을 받고 다람이가 칭찬해줘요
   - 잘 못 썼으면 "🐿️ 다시 보기" 버튼으로 획순 다시 학습 가능

---

### 🪄 글자 만들기 — 자모 합성의 비밀!

**한글의 핵심 원리를 직관적으로 배워요.**

- "ㄱ + ㅏ" 두 글자가 합쳐지면 어떻게 될까? → **"가"**!
- 받침이 붙으면 어떻게 바뀔까? → **"산"**, **"곰"**, **"달"**

3단계 흐름:

1. **🐿️ 보여주기** — `ㄱ + ㅏ = ?` (다람이가 자모 소리 들려줌)
2. **🪄 합쳐보기** — 두 자모가 미끄러져 합쳐지는 애니메이션
3. **🤔 문제 풀기** — 3개 보기 중 정답 고르기

> 💡 **왜 중요한가요?** 한글을 통째로 외우지 않고, **자모(자음+모음)가 합쳐져서 글자가 된다는 원리**를 알면 처음 보는 글자도 읽고 쓸 수 있어요.

---

### 🎮 듣고 맞추기 — 사고형 게임

**다람이가 들려주는 소리를 듣고 글자를 맞춰요.**

3가지 모드:
- **자음** — 다람이가 "기역!" 하면 ㄱ 고르기
- **모음** — 다람이가 "아!" 하면 ㅏ 고르기
- **그림** — 🐕 그림 보고 "개" 글자 고르기

**비슷하게 생긴 글자끼리** 보기로 나와서 헷갈리기 쉬운 것들을 자연스럽게 구분하게 됩니다.
- ㄱ ↔ ㅋ, ㅏ ↔ ㅓ, ㅂ ↔ ㅍ 등

오답이어도 한 번 더 들어볼 수 있어요. 너무 화나지 않게~ 🐿️

---

### 📖 문장 읽기 — 단어에서 문장으로

**"아빠가 와요!", "산이 높아요!"** 같은 짧은 문장 8개.

두 가지 모드:
- **🔊 다람이 읽기** — 다람이가 어절별로 강조하며 읽어줌
- **✋ 내가 읽기** — 아이가 단어를 순서대로 눌러 직접 읽기

> 💡 **표준 띄어쓰기**로 어절 단위(`아빠가` / `와요`)로 나뉘어 있어서, 잘못된 띄어쓰기가 학습되지 않아요.

---

### 🔄 복습하기 — 망각곡선 대응

**다람이가 "다시 보자!"고 한 글자만 모아둔 코너.**

자동으로 들어가는 글자:
- 🟠 **틀린 적이 있는 글자** — "다시 도전!" 표시
- 🟡 **복습할 시간이 된 글자** — 5분/30분/1일/2일/4일/7일 간격
  (망각곡선 기반 자동 추출)

복습 글자가 0개면 "참 잘하고 있어요!" 🎉

## 👨‍👩‍👧 부모님 모드

### 진입 방법

1. 홈 화면 아래 **"👨‍👩‍👧 부모님"** 버튼 누르기
2. 간단한 산수 문제 풀기 (아이가 우회하지 못하도록)
3. 보호자 대시보드 진입

### 대시보드에서 볼 수 있는 것

#### 👨‍👩‍👧 프로필 (형제자매 모드)
- 여러 아이를 한 기기에서 사용 가능
- 각 아이의 진도는 따로 저장
- **"+ 새 프로필"** 으로 형제용 슬롯 추가
- 칩을 클릭해 활성 프로필 전환

#### 📊 학습 요약
- ⭐ 총 별 개수
- 🎨 받은 스티커
- 🔄 복습이 필요한 글자 수

#### ⚠ 어려워하는 글자 TOP 3
- 가장 많이 틀린 글자 3개
- **💡 오프라인 활동 추천** — 예: "냉장고 자석으로 '곰' 만들기", "곰으로 시작하는 단어 5개 함께 말해보기"
- 디지털 학습 후 부모가 함께할 수 있는 활동 가이드

#### 📚 챕터별 진도
- 6개 단계별 진도 막대
- 각 단계 옆 **"초기화"** 버튼 (필요시)

#### ⚙ 설정
- **아이 이름** — 다람이가 부를 이름 (기본: 서연)
- **TTS 음성 안내** 켜기/끄기
- **효과음** 켜기/끄기

#### 🗑 초기화
- 전체 학습 데이터 초기화 (별/스티커/진도 모두 삭제)

6세 아동의 집중 시간을 고려한 자동 안내입니다.

## 💡 학습 팁 (부모님께)

### 1. 하루 사용 권장 시간
- **5세**: 10~15분
- **6세**: 15~20분
- **7세**: 20~30분
- 한 번에 길게보다 짧게 자주가 효과적입니다

### 2. 권장 학습 순서
1. **첫 주**: 준비 단계 (선 긋기) — 펜/손가락 조작 익히기
2. **2~3주**: 자음 14개 — 이름과 음가 함께 익히기
3. **4주차**: **글자 만들기** 챕터로 합성 원리 직관 이해
4. **이후**: 모음 → 쉬운 음절 → 받침 → 이중모음 순서로

### 3. 오프라인 활동 함께 하기
앱 학습은 도구일 뿐. 진짜 학습은 **삶 속**에서 일어납니다.

- 📦 **간식 봉지에서 글자 찾기** — "여기서 ㄱ 찾아볼래?"
- ✏️ **이름 쓰기** — 아이 이름의 자음을 분해해서 함께 써보기
- 📖 **그림책 읽기** — 앱과 종이책을 번갈아
- 🍎 **냉장고 자석** — 보호자 대시보드 추천 활용

### 4. 다람이의 칭찬을 100% 믿지 마세요
앱은 **글자 모양**만 평가합니다. 발음·이해도는 부모가 확인해주세요.
- "이게 무슨 글자야?" 물어보기
- "ㄱ으로 시작하는 단어 말해볼래?"

## ❓ 자주 묻는 질문

### Q. 인터넷 없이도 쓸 수 있나요?
A. 네! 한 번 접속하면 자동으로 저장되어 오프라인에서도 학습 가능합니다.

### Q. 진도가 사라지면 어떻게 하나요?
A. 같은 브라우저/기기에서는 자동 저장됩니다. 다른 기기로 옮기려면 부모님 모드에서 진도를 다시 시작해야 합니다.

### Q. 형제가 같이 써도 되나요?
A. 부모님 모드 → **"+ 새 프로필"**로 각자 슬롯을 만들면 진도가 따로 저장됩니다.

### Q. 음성이 너무 기계적이에요
A. 자연스러운 한국어 여성 음성(Microsoft Neural TTS)을 사용합니다. 만약 기계 음성이 나온다면 일부 멘트가 캐시되지 않은 경우이며, 시간이 지나면 점점 개선됩니다.

### Q. 글자를 잘 인식하지 못해요
A. 손가락보다 **터치펜/S Pen**이 정확합니다. 또한 글자를 **칸 안에 크게** 그려야 인식이 잘 됩니다.

### Q. 아이가 자꾸 별만 받으려고 해요
A. 빠르게 대충 그리면 0별이 나오고, **"🐿️ 다시 보기"** 버튼으로 획순을 다시 보게 됩니다. 보상 시스템이 인플레이션 없이 균형잡혀 있어요.

### Q. 보호자 산수 문제를 아이가 풀어요
A. 산수 문제는 5세 이상이면 풀 수 있는 수준입니다. 더 어려운 잠금이 필요하면 의견 주세요.
